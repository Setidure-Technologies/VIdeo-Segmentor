## Objectives
* Understand how to read the main scale on a measuring instrument.
* Identify the correct unit of measurement for the main scale.
* Learn to distinguish between different parts of the scale.

## Notes
The main scale is the primary scale on a measuring instrument that provides the main measurement reading. 
* The main scale is usually the larger scale on the instrument.
* It is used to measure the primary dimension of an object.
* The unit of measurement is typically indicated on the scale.

## Definitions
* **Main Scale**: The primary scale on a measuring instrument that provides the main measurement reading.
* **Unit of Measurement**: The standard unit used to express the measurement, such as millimeters (mm).

## Practical Application
The ability to read the main scale is crucial in various fields, including:
* Engineering: for precise measurement of components and assemblies.
* Manufacturing: for quality control and inspection.
* Science: for accurate data collection and experimentation.
